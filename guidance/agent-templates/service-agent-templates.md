# Service Agent Templates

Dynamics 365 Customer Service provides pre-built AI agent templates that help customers resolve routine issues instantly while empowering service teams with automation and AI assistance.

## Available agent templates

| Agent | Category | Key benefit |
|-------|----------|-------------|
| [Case Categorization & Enrichments Agent](#case-categorization) | Automation | Automated case field population (type, priority, category, product, custom fields) |
| [Automate Routine Queries Agent](#automate-routine-queries) | Self-Service | Instant 24/7 self-service across all channels |
| [After Hours Service Agent](#after-hours-service) | Support | 24/7 availability without staffing night shifts |
| [Q&A Generative and Verbatim Responses Agent](#qa-generative-verbatim) | Compliance | AI-powered question interpretation with exact verbatim responses |
| [Generate Customer Response Letter Agent](#complaint-response-letter) | Documentation | Automated FRL draft generation from Dynamics 365 case data |
| [Automated Case Follow-ups Agent](#automated-case-followups) | Automation | Automated follow-up email generation and sending |
| [Service Quality Evaluation Agent](#service-quality-evaluation) | Quality | Autonomous scoring of cases and conversations with AI-driven assessments |
| [Financial Health / Eligibility Pre-Screening Agent](#eligibility-prescreening) | Compliance | Automated information and document collection with validation |
| [Service Level Adjustment Agent](#service-level-adjustment) | Support | Real-time vulnerability signal detection from conversations and case data |
