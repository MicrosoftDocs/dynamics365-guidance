# Agent for inventory fulfillment

***Applies to***: ***Dynamics 365 Commerce***

An agent that interprets fulfillment requests, checks multi-site availability, available to promise (ATP), and Inventory Visibility Service (IVS), recommends the best fulfillment option by policy, and uses Dynamics 365 capabilities to create sales orders and transfer orders with optional soft reservations. The agent is available across points of sale (POS), Teams, and call-center applications.

> [!NOTE]
> The agent templates aren't yet ready for download. If you're interested in the commerce agent templates, [fill in this form](https://aka.ms/commerce-agent-templates).

## Overview

The agent provides an instant overview of availability and promise-to-deliver answers. Then, it runs the chosen fulfillment path (order or transfer) with traceability, which reduces walk-aways and manual effort.  

## Key benefits

- Faster ATP and fulfillment decisions across sites​
- Fewer lost sales at the point of service​
- Less system-hopping; one guided flow end-to-end​
- Traceable execution (SO/TO, optional reservations)​
- Extensible + low-code orchestration (Copilot Studio)

## Scenarios

- In-store promise, transfer, and sale in one flow​
- Source from DC or other stores when local is out​
- Fulfillment by vendor or external provider where enabled​
- Time-bound soft reservations for named customers​
- Consistent outcomes across channels

## Industries
