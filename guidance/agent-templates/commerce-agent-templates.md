# Commerce agent templates

***Applies to***: ***Dynamics 365 Commerce***, ***Dynamics 365 Supply Chain Management***

Commerce agents transform retail operations by automating reconciliation tasks, inventory fulfillment, store health monitoring, and skills compliance at scale. Purpose‑built AI agents proactively detect problems, execute the right actions, and enforce policies across Dynamics 365 Commerce systems. They reduce exceptions, accelerate resolution, and keep stores running smoothly. The result is resilient, intelligent Commerce operations powered by automation, not manual effort.

> [!NOTE]
> The agent templates aren't yet available for download. If you're interested in the commerce agent templates, [fill in this form](https://aka.ms/commerce-agent-templates).

## Available agent templates

These commerce agents use Dynamics 365 Commerce capabilities, alongside Microsoft Copilot Studio and Copilot in Dynamics 365.

| Agent | Category | Key benefit |
| --------- | ------------ | ------------- |
| [Payment Reconciliation Agent](payment-reconciliation-agent.md) | Automation | Automated reconciliation of retail payments. |
| [Inventory Fulfillment Agent](inventory-fulfillment-agent.md) | Self-Service, Automation | Provides instant, consistent availability and promise-to-deliver answers, then executes the chosen fulfillment path (order/transfer) with traceability. |
| [Store Monitoring Agent](store-monitoring-agent.md) | Self-Service | Provides a single, proactive, conversational control plane for monitoring thousands of retail POS devices. |
| [Adaptive Skills Policy Agent](adaptive-skills-policy-agent.md) | Automation, Compliance | A configurable agent that uses an approved set of processes and policies to plan and run business tasks through the [Model Context Protocol for finance and operations apps](/dynamics365/fin-ops-core/dev-itpro/copilot/copilot-mcp). |
